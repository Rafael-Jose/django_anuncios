Site de anúncios criado usando Django. Bem semelhante ao projeto de notícias, porém nesse site implementei o uso de CSS e um pouco de JavaScript.
Além de deixar o campo de pesquisa dinâmico, onde ele encontrará o anúncio de nome exato ao da pesquisa.
Utilizei o Framework Fomantic-UI, que se refere ao Semantic, porém numa versão mais nova.
O banco de dados do site foi o sqlite3 por ser mais simples de ser usado, diferente de outros databases onde é preciso criar um host etc.
É possível entrar na aba de admin, seguindo o link de localhost/admin, para testes recomendo criar o superuser como login = admin e senha = admin. Na aba admin tem-se o acesso para criar categorias e, posteriormente, criar anúncios para essas categorias.
Na parte Lateral da Home existe um menu, o qual filtra a categoria de anúncios desejada.
